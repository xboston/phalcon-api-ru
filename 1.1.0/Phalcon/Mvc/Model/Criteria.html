Class Phalcon\Mvc\Model\Criteria

Namespace: Phalcon\Mvc\Model
Implements: Phalcon\Mvc\Model\CriteriaInterface, Phalcon\DI\InjectionAwareInterface,
Phalcon\Mvc\Model\Criteria This class allows to build the array parameter required by Phalcon\Mvc\Model::find and Phalcon\Mvc\Model::findFirst, using an object-oriented interfase
