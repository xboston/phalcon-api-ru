Class Phalcon\Http\Response

Namespace: Phalcon\Http
Implements: Phalcon\Http\ResponseInterface, Phalcon\DI\InjectionAwareInterface,
Phalcon\Http\Response Part of the HTTP cycle is return responses to the clients. Phalcon\HTTP\Response is the Phalcon component responsible to achieve this task. HTTP responses are usually composed by headers and body.
	$response = new Phalcon\Http\Response();
	$response->setStatusCode(200, "OK");
	$response->setContent("Hello");
	$response->send();
