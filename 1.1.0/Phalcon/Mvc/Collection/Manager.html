Class Phalcon\Mvc\Collection\Manager

Namespace: Phalcon\Mvc\Collection
Implements: Phalcon\DI\InjectionAwareInterface, Phalcon\Events\EventsAwareInterface,
Phalcon\Mvc\Collection\Manager This components controls the initialization of models, keeping record of relations between the different models of the application. A CollectionManager is injected to a model via a Dependency Injector Container such as Phalcon\DI.
 $di = new Phalcon\DI();

 $di->set('collectionManager', function(){
      return new Phalcon\Mvc\Collection\Manager();
 });

 $robot = new Robots($di);
