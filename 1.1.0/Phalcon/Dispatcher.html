Class Phalcon\Dispatcher

Namespace: Phalcon
Implements: Phalcon\DispatcherInterface, Phalcon\DI\InjectionAwareInterface, Phalcon\Events\EventsAwareInterface,
Phalcon\Dispatcher This is the base class for Phalcon\Mvc\Dispatcher and Phalcon\CLI\Dispatcher. This class can't be instantiated directly, you can use it to create your own dispatchers
